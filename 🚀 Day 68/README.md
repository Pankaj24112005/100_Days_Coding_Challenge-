## Overview

On Day 68, I focused on understanding the theoretical foundations of **Linear Regression**, one of the most fundamental algorithms in Machine Learning.

Linear Regression is a supervised learning algorithm used to predict continuous values by modeling the relationship between independent variables (X) and a dependent variable (Y).

---

## Simple Linear Regression

Formula:

y = mx + c

Where:
- m → Slope (coefficient)
- c → Intercept
- x → Input feature
- y → Predicted output

---

## Multiple Linear Regression

Formula:

y = b0 + b1x1 + b2x2 + ... + bnxn

Used when there are multiple input features.

---

## Cost Function

The most commonly used cost function is **Mean Squared Error (MSE)**.

MSE = (1/n) Σ (Actual - Predicted)^2

Goal: Minimize the error.

---

## Gradient Descent

Gradient Descent is an optimization algorithm used to minimize the cost function.

Steps:
1. Initialize parameters
2. Compute error
3. Update parameters using learning rate
4. Repeat until convergence

---

## Evaluation Metrics

- R² Score
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)

---

## Assumptions of Linear Regression

1. Linearity  
2. Independence of errors  
3. Homoscedasticity  
4. Normal distribution of residuals  

---
