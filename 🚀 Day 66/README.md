# Day 66 – Machine Learning Types & Workflow

Part of my **#100DaysCodingChallenge**

This project focuses on understanding the **types of Machine Learning** and the **complete ML workflow**, along with a simple hands-on implementation using the Iris dataset.

---

## Topics Covered

### Types of Machine Learning

- Supervised Learning  
- Unsupervised Learning  
- Semi-Supervised Learning  
- Reinforcement Learning  

---

### Typical Machine Learning Workflow

1. Problem Understanding  
2. Data Collection  
3. Data Preprocessing  
4. Exploratory Data Analysis (EDA)  
5. Model Selection  
6. Model Training  
7. Model Evaluation  
8. Deployment  

---

## Files Included

- `Day66_ML_Types_and_Workflow.ipynb` – Jupyter Notebook with:
  - Theory on ML Types
  - ML Workflow explanation
  - Iris dataset loading
  - Train-test split
  - Logistic Regression model
  - Accuracy evaluation

---

## Objective

To understand how Machine Learning projects move from raw data to trained models and how different ML types are applied based on problem statements.

---

## Libraries Used

- scikit-learn  
- numpy  

---

## Output

A trained Logistic Regression model with accuracy evaluation on the Iris dataset.

---

## Conclusion

This mini project helped me understand the end-to-end ML pipeline and clarified when to use different Machine Learning approaches. Consistency is key in mastering Data Science and AI.
