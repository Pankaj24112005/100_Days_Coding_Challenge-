## 📌 What is Random Forest?

Random Forest is a **supervised machine learning algorithm** used for both **classification and regression** tasks. It works by creating multiple Decision Trees during training and combining their outputs to produce a more accurate and stable prediction.

Key Idea:
- Multiple Decision Trees are built
- Each tree makes a prediction
- Final output is determined by **majority voting (classification)** or **averaging (regression)**
