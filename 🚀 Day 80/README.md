## 📌 What is Cross Validation?

Cross Validation is a model evaluation technique where the dataset is divided into multiple parts and the model is trained and tested multiple times.

Instead of using only a single **train-test split**, Cross Validation ensures that the model is evaluated on different subsets of the data.

This helps produce **more reliable and stable performance results**.
