## 📌 Topic: K-Means Clustering

K-Means is an **unsupervised machine learning algorithm** used to group similar data points into clusters.  
It works by finding the **centroids (center points)** of clusters and assigning each data point to the nearest centroid.

## ⚙️ Steps in K-Means Algorithm

1. Choose the number of clusters **K**
2. Initialize **K centroids** randomly
3. Assign each data point to the **nearest centroid**
4. Recalculate the **centroid of each cluster**
5. Repeat the process until the centroids stop changing
