## 📌 What is Hyperparameter Tuning?

Hyperparameters are the configuration settings that are defined **before training a machine learning model**.
Unlike model parameters, hyperparameters are not learned from the data and must be **manually set or optimized**.

Hyperparameter tuning helps find the **best combination of parameters** that improves model accuracy and generalization.

---

## ⚙️ Techniques for Hyperparameter Tuning

### 1️⃣ Grid Search

Grid Search tries **all possible combinations** of hyperparameters and selects the best one.

### 2️⃣ Random Search

Random Search selects **random combinations of hyperparameters**, which can be faster than Grid Search.

### 3️⃣ Bayesian Optimization

An advanced method that uses **probability and past results** to find the best parameters efficiently.
