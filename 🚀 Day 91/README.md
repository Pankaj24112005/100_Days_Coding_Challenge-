🚀 Day 91 – Streamlit Basics | Iris Prediction App

Project Overview
This project is part of my #100DaysOfCode Challenge.
I built a Machine Learning web app using Streamlit to predict Iris flower species..

Input Features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Output:
- Setosa 🌸
- Versicolor 🌼
- Virginica 🌺

Key Learnings:
- Built interactive UI using Streamlit
- Loaded ML model using Pickle
- Understood encoded outputs (0,1,2 → labels)

Tech Stack:
- Python
- NumPy
- Scikit-learn
- Streamlit
- Pickle

How to Run:

Step 1: Install dependencies
pip install streamlit scikit-learn numpy

Step 2: Run Streamlit app
streamlit run app.py

Step 3: Show README in terminal
python readme_terminal.py

Example Input:
Sepal Length: 5.1
Sepal Width : 3.5
Petal Length: 1.4
Petal Width : 0.2

Output: Setosa 🌸
