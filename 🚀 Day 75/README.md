# 🚀 Day 75 – K-Nearest Neighbors (KNN)

This project is part of my **#100DaysOfCoding Challenge** where I am learning **Data Science and Machine Learning step by step**.

## 📌 Topic
K-Nearest Neighbors (KNN)

KNN is a **supervised machine learning algorithm** used for **classification and regression**. It works by finding the **K closest data points (neighbors)** and making predictions based on their labels.

## 🧠 What I Learned
- Concept of **K-Nearest Neighbors**
- Importance of **K value**
- Distance metrics (Euclidean Distance)
- Training a **KNN model using Scikit-learn**
- Evaluating model accuracy

## 📊 Dataset
I used the **Breast Cancer Dataset** available in **Scikit-learn**.

Features: Medical measurements of tumors  
Target:
- 0 → Malignant
- 1 → Benign

## ⚙️ Technologies Used

- Jupyter Notebook

## 📈 Steps Performed
