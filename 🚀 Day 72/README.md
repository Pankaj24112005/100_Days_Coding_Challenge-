Day 72 – Logistic Regression (Theory)
--------------------------------------------------

Overview:
Logistic Regression is a supervised Machine Learning
algorithm used for binary classification problems.

It predicts probability using the Sigmoid function
and classifies output based on a threshold (0.5).

Topics Covered

2. Linear vs Logistic Regression

4. Log Loss (Binary Cross-Entropy)
5. Decision Boundary
6. Assumptions
7. Advantages & Limitations


--------------------------------------------------
Key Mathematical Formulas
--------------------------------------------------

Sigmoid Function:
sigma(z) = 1 / (1 + e^(-z))

Where:
z = w0 + w1x1 + w2x2 + ... + wnxn


Log Loss:
Loss = - (1/m) * Σ [ y log(y_hat)
                    + (1 - y) log(1 - y_hat) ]
