# Day 70 - Model Evaluation Metrics

This notebook is part of my 100 Days of Code journey focused on building strong Machine Learning fundamentals.

------------------------------------------------------------

Objective:
To understand and implement important evaluation metrics for
both Classification and Regression models.

Building a model is easy.
Evaluating it properly is what makes it reliable.

------------------------------------------------------------

Classification Metrics Covered:

- Confusion Matrix
- Accuracy
- Precision
- Recall
- F1 Score
- ROC Curve
- AUC Score

Key Learnings:

- Accuracy alone can be misleading in imbalanced datasets.
- Precision is important when False Positives are costly.
- Recall is important when False Negatives are dangerous.
- F1 Score balances Precision and Recall.
- ROC-AUC gives an overall measure of model performance.

------------------------------------------------------------

Regression Metrics Covered:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)

Key Learnings:

- MAE measures average prediction error.
- MSE penalizes larger errors more heavily.
- RMSE gives error in the original scale of the target.
