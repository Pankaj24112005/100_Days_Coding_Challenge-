⏰ Digital Clock using Python (Tkinter)

This is a GUI-based Digital Clock built using Python and Tkinter as part of my Day 21 of the 100 Days of Code Challenge.

The clock displays the current time (HH:MM:SS AM/PM) along with the current date, and updates automatically every second.

Features

Real-time digital clock

Displays time in HH:MM:SS AM/PM format

Shows current date

Auto-refreshes every 1 second

Simple and beginner-friendly GUI

Built using Python’s built-in libraries

Technologies Used

Python

Tkinter (for GUI)

time.strftime() (for formatting time & date)
