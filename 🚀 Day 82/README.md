## 📌 What is Clustering?

Clustering is an **unsupervised machine learning technique** used to group similar data points together.

The goal of clustering is to:
- Place similar data points in the same group
- Separate dissimilar data points into different groups

---

## 📊 Why Clustering is Important

Clustering helps in discovering **hidden patterns in data**.

It is widely used in:
- Customer Segmentation
- Recommendation Systems
- Image Segmentation
- Anomaly Detection

---

## 🔍 Supervised vs Unsupervised Learning

| Type | Description |
|-----|-------------|
| Supervised Learning | Data has labels |
| Unsupervised Learning | Data has no labels |

Clustering belongs to **Unsupervised Learning**.
