# 🚀 Day 97 — ML Interview Preparation

## 📌 Topics Covered

### 1️⃣ Bias vs Variance
- **Bias** → Error due to overly simple assumptions in the model.
- **Variance** → Error due to model sensitivity to training data.
- Goal: Find the right balance between both.

---

### 2️⃣ Overfitting
- Happens when a model learns training data too well.
- Performs great on training data but poorly on new/unseen data.
- Can be reduced using:
  - Cross Validation
  - Regularization
  - More Data
  - Simpler Models

---

### 3️⃣ Precision vs Recall

| Metric | Meaning |
|--------|---------|
| Precision | Out of predicted positives, how many are correct |
| Recall | Out of actual positives, how many were identified |

### Formula:
- Precision = TP / (TP + FP)
- Recall = TP / (TP + FN)

---

### 4️⃣ Random Forest
- Ensemble learning algorithm.
- Uses multiple decision trees.
- Final output is based on majority voting.
- Advantages:
  - High Accuracy
  - Reduces Overfitting
  - Works well on large datasets

---

### 5️⃣ Cross Validation
- Technique used to evaluate model performance.
- Dataset is split into multiple parts (folds).
- Helps improve model reliability.
