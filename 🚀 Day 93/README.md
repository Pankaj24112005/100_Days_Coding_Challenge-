## 💻 Important Git Commands

### Initialize Git

```bash
git init
```

### Add Files

```bash
git add .
```

### Commit Changes

```bash
git commit -m "commit message"
```

### Connect GitHub Repository

```bash
git remote add origin <repository_link>
```

### Push Code

```bash
git push -u origin main
```

### Pull Updates

```bash
git pull origin main
```
