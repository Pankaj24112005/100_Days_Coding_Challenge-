## 📌 What is Hierarchical Clustering?

Hierarchical Clustering builds a tree-like structure (called a **dendrogram**) to represent data grouping.

It does not require specifying the number of clusters initially, making it different from K-Means.

---

## 🔍 Types of Hierarchical Clustering

### 1️⃣ Agglomerative (Bottom-Up)
- Start with each data point as its own cluster  
- Merge the closest clusters step-by-step  

### 2️⃣ Divisive (Top-Down)
- Start with one cluster  
- Split it recursively into smaller clusters  

---

## 📊 Dendrogram

A **dendrogram** is a tree diagram used to visualize how clusters are formed.

It helps in:
- Understanding cluster relationships  
- Deciding the optimal number of clusters

## 💡 Key Learnings

- Hierarchical clustering builds clusters step-by-step  
- Dendrogram plays a crucial role in cluster selection  
- Works well for smaller datasets  
- No need to predefine number of clusters  

---

## ⚠️ Limitations

- Computationally expensive for large datasets  
- Sensitive to noise and outliers
