# 🤖 Common ML Interview Questions

## 20 Frequently Asked ML Interview Questions

### 1. What is Machine Learning?
Machine Learning is a branch of AI that allows systems to learn from data and improve automatically without explicit programming.

---

### 2. Difference between AI, ML, and Deep Learning?
- AI → Broad concept of intelligent machines
- ML → Subset of AI that learns from data
- Deep Learning → Subset of ML using neural networks

---

### 3. What is Supervised Learning?
A type of learning where models are trained using labeled data.

Example:
- Spam Detection
- House Price Prediction

---

### 4. What is Unsupervised Learning?
Learning using unlabeled data to find patterns or clusters.

Example:
- Customer Segmentation
- Clustering

---

### 5. What is Reinforcement Learning?
An agent learns by interacting with an environment using rewards and penalties.

---

### 6. What is Overfitting?
When a model performs well on training data but poorly on unseen data.

---

### 7. What is Underfitting?
When a model fails to learn patterns from training data.

---

### 8. Difference between Classification and Regression?
- Classification → Predict categories
- Regression → Predict continuous values

---

### 9. What is Bias in ML?
Error caused by overly simple assumptions in the learning algorithm.

---

### 10. What is Variance in ML?
Error caused by too much sensitivity to training data.

---

### 11. What is the Train-Test Split?
Dividing data into training and testing sets for evaluation.

Common ratio:
- 80:20
- 70:30

---

### 12. What is Cross Validation?
A technique used to evaluate model performance on different subsets of data.

---

### 13. What is Accuracy?
The percentage of correctly predicted observations.

---

### 14. What are Precision and Recall?
- Precision → Correct positive predictions out of total predicted positives
- Recall → Correct positive predictions out of actual positives

---

### 15. What is F1-Score?
Harmonic mean of Precision and Recall.

---

### 16. What is a Confusion Matrix?
A table used to evaluate classification model performance.

---

### 17. What is Feature Engineering?
The process of selecting, modifying, or creating useful input features.

---

### 18. What is Gradient Descent?
An optimization algorithm used to minimize model error.

---

### 19. What is the difference between Bagging and Boosting?
- Bagging → Parallel learning
- Boosting → Sequential learning focusing on mistakes

---

### 20. Explain one ML project you have worked on.
This question tests:
- Problem-solving ability
- Technical understanding
- Communication skills

Tip:
Explain:
1. Problem Statement
2. Dataset
3. Preprocessing
4. Model Used
5. Accuracy/Results
6. Challenges Faced
