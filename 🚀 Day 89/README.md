# Heart Disease Prediction (Classification Project).

**Day 89 – #100DaysOfCode**

This project focuses on building a **Machine Learning classification model** to predict whether a person is at risk of heart disease based on medical attributes.

---

## Problem Statement
Heart disease is one of the leading causes of death worldwide. Early prediction can help in timely diagnosis and treatment.  

The goal of this project is to develop a model that can **classify patients into risk categories** based on clinical features.

---

## Dataset Information
The dataset contains various medical attributes such as:

- Age  
- Sex  
- Chest Pain Type  
- Resting Blood Pressure  
- Cholesterol Level  
- Fasting Blood Sugar  
- Rest ECG  
- Maximum Heart Rate  
- Exercise Induced Angina  
- ST Depression  
- Target (0 = No Disease, 1 = Disease)
