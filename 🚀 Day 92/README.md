# ❤️ Heart Disease Prediction System.

A Machine Learning project that predicts the possibility of heart disease using patient medical data.

## 🚀 Project Overview

This project uses Machine Learning algorithms to analyze health parameters and predict whether a person is likely to have heart disease or not.

The model is trained on a heart disease dataset and deployed using Python.

---

## 📌 Features

- Heart disease prediction using ML
- Data preprocessing and scaling
- Trained machine learning model
- Easy-to-use prediction system
- Jupyter Notebook for model training
- Deployment-ready structure

---

## 📊 Dataset

The dataset contains medical attributes such as:

- Age
- Sex
- Chest Pain Type
- Blood Pressure
- Cholesterol
- Maximum Heart Rate
- ECG Results
- Exercise Induced Angina
- Oldpeak
- ST Slope




