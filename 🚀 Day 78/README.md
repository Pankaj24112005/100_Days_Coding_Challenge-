---

## 📚 What I Learned

### 🔹 Underfitting

Underfitting occurs when a model is **too simple** to capture the underlying patterns in the dataset.

**Characteristics:**

* High training error
* High testing error
* Model fails to learn the data patterns

**Common Causes:**

* Model too simple
* Insufficient training
* Not enough features

---

### 🔹 Overfitting

Overfitting occurs when a model learns the **training data too well**, including noise and small variations.

**Characteristics:**

* Very low training error
* High testing error
* Poor generalization to new data

**Common Causes:**

* Model too complex
* Too many features
* Small training dataset

---

## ⚖️ Bias–Variance Tradeoff

A good Machine Learning model should balance:

* **High Bias → Underfitting**
* **High Variance → Overfitting**

The goal is to achieve a **model that generalizes well on unseen data**.
