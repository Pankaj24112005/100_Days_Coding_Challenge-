# 🚀 Day 76 – Decision Trees

This project is part of my **#100DaysOfCoding Challenge**, where I am learning and implementing Machine Learning concepts step by step.

## 📌 Topic

**Decision Tree Classifier**

Decision Trees are supervised machine learning algorithms used for **classification and regression tasks**.
They work by splitting the dataset into branches based on feature values, forming a tree-like structure that helps make predictions.

## 📊 Dataset

In this project, I used the **Breast Cancer dataset** from `sklearn.datasets`.

Features include medical measurements used to classify tumors as:

* **Malignant**
* **Benign**

## ⚙️ Technologies Used

* Python
* NumPy
* Pandas
* Scikit-learn
* Jupyter Notebook

## 🧠 Steps Implemented

1. Import required libraries
2. Load the dataset
3. Split the dataset into training and testing sets
4. Train the **Decision Tree Classifier**
5. Make predictions on test data
6. Evaluate the model using:

   * Accuracy Score
   * Confusion Matrix
   * Classification Report

## 📈 Model Performance

The model predicts whether a tumor is **malignant or benign** based on medical features.

Evaluation metrics used:

* **Accuracy**
* **Confusion Matrix**
* **Precision, Recall, F1-score**
